Sistema de Control y Verificación

Academia COHAB

Sistema institucional de gestión de suscripciones y control de acceso. Transparencia, orden y disciplina en cada validación.

Cómo Funciona el Sistema

El proceso de validación es simple, rápido y preciso. Cada paso está diseñado para garantizar el orden y la transparencia en el acceso a las instalaciones.

1
Pago de Suscripción El alumno realiza el pago de su mensualidad. El sistema registra automáticamente la fecha de pago y calcula el período de validez.
2
Registro del Estado El sistema actualiza el estado de la suscripción en tiempo real. Cada alumno tiene un estado binario: ACTIVA o VENCIDA.
3
Verificación en Entrada El profesor escanea el código QR del alumno o ingresa su ID. El sistema consulta la base de datos y determina el estado actual.
4
Control de Acceso El sistema responde de forma inmediata: Acceso permitido o Acceso denegado. La asistencia queda registrada automáticamente.

Criterio de validación: Una suscripción se considera ACTIVA cuando los días restantes hasta el próximo vencimiento son mayores o iguales a cero. Si los días son negativos, la suscripción está VENCIDA y el acceso queda denegado.

Roles y Permisos

El sistema está diseñado con roles claramente definidos. Cada usuario tiene acceso únicamente a la información y funciones correspondientes a su rol.

Alumno

Consulta tu estado de suscripción y mantén un registro de tus pagos.

Puedes ver:

  • Estado actual de tu suscripción
  • Fecha de último pago registrado
  • Fecha del próximo vencimiento
  • Días restantes o días de atraso
  • Historial de pagos

No puedes:

  • Ver información de otros alumnos
  • Modificar datos administrativos
  • Acceder a funciones de gestión
Consultar mi estado

Profesor

Valida el acceso de alumnos en la entrada y registra asistencia de forma rápida y precisa.

Puedes hacer:

  • Escanear código QR del alumno
  • Ingresar ID manualmente
  • Ver resultado inmediato de validación
  • Registrar asistencia automáticamente

Ves en pantalla:

  • Estado de suscripción: ACTIVA o VENCIDA
  • Información básica del alumno
  • Días restantes o días de atraso
Validar acceso

Administrador

Gestiona toda la información del sistema, alumnos, pagos y configuraciones.

Gestionas:

  • Registro y edición de alumnos
  • Registro de pagos y actualización de fechas
  • Configuración del sistema
  • Reportes y estadísticas
  • Generación de códigos QR

Nota: El acceso administrativo no está disponible desde esta página. Contacta con el administrador del sistema para obtener credenciales.

Seguridad y Privacidad

La protección de datos y la privacidad de los alumnos son prioridades fundamentales en el diseño de este sistema.

  • Almacenamiento seguro: Todos los datos se almacenan en servidores seguros con encriptación de extremo a extremo
  • Validación en tiempo real: Cada consulta se realiza directamente contra la base de datos, garantizando información siempre actualizada
  • Identificadores únicos: Cada alumno posee un ID único y privado que solo él conoce
  • Principio de menor privilegio: Cada rol tiene acceso únicamente a la información necesaria para su función
  • Sin exposición de datos: La información personal no se comparte entre usuarios ni se expone públicamente
