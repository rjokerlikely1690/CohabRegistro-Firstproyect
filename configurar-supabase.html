1 Crea tu proyecto en Supabase

  1. Ve a supabase.com
  2. Crea una cuenta gratis (con GitHub o email)
  3. Click en "New Project"
  4. Elige un nombre y contraseña
  5. Espera 2 minutos mientras se crea

2 Copia tus credenciales

  1. En tu proyecto, ve a Settings → API
  2. Copia el Project URL
  3. Copia el anon public key
💡 Tip: El "anon public" key es seguro compartir, está diseñado para el navegador

3 Pega tus credenciales aquí

4 Crea la tabla de alumnos

Después de guardar la configuración, ve a:

📄 migrar-mongodb-a-supabase.html

Para crear la tabla y migrar tus datos automáticamente
